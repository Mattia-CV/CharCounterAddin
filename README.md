# CharCounterAddIn

This little application was designed by a member of Cambiaverso to help other workmates with their Word's file

# Run in Windows

- Select ```main``` as branch
- On the root folder type ```npm start```
- Type ```Y``` in the terminal (after the question)
- Done

# Run in Linux

- Select ```ssh``` as branch
- On the root folder type ```npm run dev-server```
- Done

# View add-in in Word

- Open a Word's project
- Select the Home tab
- Click the Add-ins button (right side)
- On the menu select ```Conta i caratteri```
- Click ```OK``` on the menu
- Done
